# Sitemap Update Instructions

This document explains how to keep the sitemap.xml file up to date with your blog posts.

## Overview

The `sitemap.xml` file is located at `frontend/public/sitemap.xml` and helps search engines discover and index your content.

## When to Update

Update the sitemap whenever you:
- Add a new blog post
- Remove a blog post
- Change a post's slug/URL
